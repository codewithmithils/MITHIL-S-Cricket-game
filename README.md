Interactive Cricket Game

A small cricket game made using basic web technologies.
The game updates runs, wickets, and overs when the user clicks the buttons.

Game Instructions

Click Hit to score runs for Team 1.

Click Bowl to score runs for Team 2.

Runs or wickets appear randomly.

Overs increase after every 6 balls.

The scoreboard and commentary update after each ball.

Use Reset to start a new match.

Technologies Used

HTML — page structure

CSS — simple styling and layout

JavaScript — game logic, score updates, wickets, overs, and commentary
